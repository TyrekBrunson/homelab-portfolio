Active Directory Home Lab — Windows Server 2022
Proxmox VE Windows Server Active Directory

Deployed a full Active Directory environment from scratch on a Windows Server 2022 VM hosted on the Proxmox VE hypervisor (Alienware Alpha R1). Installed and configured Active Directory Domain Services (AD DS), promoted the server to a Domain Controller, and established a local domain. Built out a realistic organizational structure with multiple Organizational Units (OUs) modeled after enterprise environments — including geographic containers (USA, Europe, Asia) with nested OUs for Users, Computers, and Servers. Created Security Groups (both built-in and custom) with appropriate group scopes (Global/Universal/Domain Local) and group types (Security and Distribution) to reflect real-world IAM practices. Created and managed user accounts and assigned group memberships. Applied Group Policy concepts and explored role-based access control across departments (IT, Finance, HR). This lab directly mirrors the Active Directory workflows used in enterprise IT environments on a daily basis.
